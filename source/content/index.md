2. Midwest American Transportation Center (MATC) Smart Barrier - Autonomous Vehicle Research with GPS/GNSS 

PhD team research project that focused on developing algorithms for improving lane correction based on the curvature of the current road that a vehicle is occupying at the time of measurement/capture. 

My role was to be the team programmer and GPS researcher - I parsed and refactored MATLAB code into Python to be able to work with the algorithms and the GPS/GNSS modules to measure the realtime data of a moving vehicle, then send that data to a JavaScript, ArcGIS-based, Google Maps-like interface to be able to see what and how the algorithm would suggest the vehicle to correct its lane position to be center on the road.

TL;DR - Built a vehicle-tracking webapp using JavaScript, GPS, and ArcGIS REST API for real-time visualization.
